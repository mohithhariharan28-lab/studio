# **App Name**: Mohith's Portfolio

## Core Features:

- Hero Section: Two-column layout with greeting, name, summary, and CTA on the left; profile image placeholder on the right.
- About Section: Concise paragraph about aspirations in data analysis, business analysis, and entrepreneurial interests.
- Skills Showcase: Clean tag-style layout displaying skills like Data Analysis, Business Analysis, and Problem Solving.
- Projects Portfolio: Card-based grid showcasing projects with titles, descriptions, and links to GitHub repositories.
- Certifications Display: List of certifications (Wadhwani, IBM, GUVI) in a clean, professional format.
- Contact Information: Display contact details (mobile, email) and links to LinkedIn and GitHub profiles.
- Project relevance selector: Use a tool to decide which projects are most relevant to highlight on the homepage, based on user data or popularity.

## Style Guidelines:

- Background: White (#FFFFFF) for a clean look.
- Primary text: Deep navy (#191970) for readability.
- Primary accent: Classic blue (#000080) for buttons and links.
- Highlight/alert: Warm vibrant orange (#FF4500) for emphasis.
- Font: Clean, modern sans-serif (e.g., 'Inter', 'Roboto') for all text.
- Cards: White background, rounded corners, subtle shadow.
- Responsive layout for desktop and mobile.
- Minimal hover effects using classic blue or vibrant orange.
